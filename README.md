## Quick start

1.  Tải trực tiếp `.zip` hoặc clone source `Github` này về! 
2.  Cài đặt NodeJs (<https://nodejs.org/en/>)
3.  Mở Terminal trên máy của bạn và đi đến thư mục đã tải hoặc clone source sau đó gõ: `npm install` hoặc `yarn install`
4.  Chạy project bằng `npm start`.
